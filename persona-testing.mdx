## Overview

Persona Testing simulates how different user types respond to your designs. Create personas from your Figma frames, then test designs with synthetic users to get behavioral and emotional insights without recruiting anyone.

**Time per test**: ~2 minutes  
**Cost**: 50 credits per test  
**Initial setup**: 2-4 frames (one-time to create personas)  
**Test frames**: 1 frame per test  

---

## How It Works

### First Time: Create Personas

If you don't have personas yet, create them first.

**Step 1: Select 2-4 Frames**

Choose key screens from your Figma file that represent your product. More frames = better context for persona generation.

**Step 2: Generate Personas**

AI analyzes your design to create personas automatically (~45 seconds).

**Step 3: Manage Personas**

Review, edit, or add custom personas. Personas include:
- Demographics (age, gender, location, income, education)
- Professional profile (job title, industry, experience, company size)
- Goals (primary, secondary, long-term)
- Pain points (functional, emotional, social)
- Tech savviness (low, medium, high)
- Brand loyalty (low, medium, high)
- Decision-making style
- Preferred communication channels

After initial setup, you have personas and can skip to testing. Add more personas anytime via **Manage Personas**.

---

### Every Test: Test Your Designs

Once you have personas, test designs with them.

**Step 1: Select a Frame**

Pick one screen you want feedback on. Test one frame per test. Can retest the same frame with different personas or questions.

**Step 2: Ask a Question**

Ask what you want to know about user response:
- "What catches their attention first?"
- "Where do they get confused?"
- "Would they complete this task?"
- "What's missing from this design?"

500 character limit. Get AI suggestions for follow-up questions.

**Processing** (~90 seconds):
1. Analyzing design context (15s)
2. Simulating persona perspective(s) (15s)
3. Generating persona response(s) (15s)
4. Evaluating feedback quality (15s)
5. Finalizing responses (15s)

## Understanding Results

### What You Get

For each persona tested:
- **Direct response** - What the persona would do or say
- **Rationale** - Why they respond that way
- **Behavioral insight** - How their traits influence their response

### Multi-Persona vs Single-Persona

**Multi-persona test** - Get all personas' responses in one test:
- Identify consensus (what everyone agrees on)
- Spot disagreement (where personas differ)
- See shared pain points
- Find gaps everyone identifies

**Single-persona test** - Deep dive on one persona:
- Focused perspective
- Compare multiple single tests to see differences

## Managing Personas

### After Initial Setup

From **Manage Personas** button:

**View & Edit**
- See all persona details (demographics, goals, pain points, etc.)
- Edit any attribute anytime
- Changes apply to future tests immediately

**Add Custom Personas**
- Create personas manually from scratch
- No frames required
- Combine with auto-generated personas

**Delete Personas**
- Remove personas you don't need
- Deletion is permanent
- Existing test results preserved

## FAQ

<Accordion title="How many personas can I create?">
Unlimited. Create as many as needed. Test with all at once or individually.
</Accordion>

<Accordion title="Do I need to set up personas with frames every time?">
No. The 2-4 frame setup is one-time only. After you create personas, you can test designs without adding more frames. You can add custom personas anytime without frames.
</Accordion>

<Accordion title="Can I edit personas after creating them?">
Yes. Go to **Manage Personas** to edit any persona. Changes apply to future tests immediately.
</Accordion>

<Accordion title="Can I test the same frame with different personas?">
Yes. Run multiple tests with the same frame but different personas or questions.
</Accordion>

<Accordion title="What if I get a rate limit error on suggestions?">
The suggestions endpoint has rate limiting. Wait a few minutes and try again. You can still test without suggestions.
</Accordion>

<Accordion title="Do I need real user research?">
No. Personas are synthetic simulations based on AI. They don't replace user research but provide rapid feedback for iteration.
</Accordion>

<Accordion title="What's the difference between tech savviness levels?">
Each persona has a tech level (low, medium, high) which influences how they respond to technical features and complexity.
</Accordion>

<Accordion title="How many credits does a test cost?">
Each test costs **50 credits**, regardless of how many personas you test.
</Accordion>

<Accordion title="Can I share test results?">
Save and reference test results from your project history. Share through your team workspace.
</Accordion>
